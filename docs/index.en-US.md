## Let’s Start

你可以通过以下命令快速启动开发服务：

<br/>

```bash
yarn create react-app my-app --template doly # 创建一个应用

cd my-app

yarn add husky@4 --dev # 集成 commitlint

yarn start # 启动本地开发服务
```

<br/>

_或者你使用的是 `npm` :_

<br/>

```bash
npx create-react-app my-app --template doly # 创建一个应用

cd my-app

npm install husky@4 --save-dev # 集成 commitlint

npm start # 启动本地开发服务
```
